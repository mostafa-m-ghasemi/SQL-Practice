# SQL Mastery Roadmap 🗄️

Welcome to my SQL learning journey! This repository documents my progress in mastering Structured Query Language (SQL) with a current focus on **PostgreSQL** and future plans to expand to other database systems.

## 🛠️ Tech Stack & Tools
- **Current Focus:** PostgreSQL
- **Future Goals:** SQLite, SQL Server, MySQL
- **Environment:** Jupyter Notebook / JupyterLab
- **Language:** Python (for executing queries and data visualization)

## 📚 Repository Structure
The projects are organized by skill level and topic:

1.  **🟢 Foundations:**
    *   Basic `SELECT`, `WHERE`, and `ORDER BY` clauses.
    *   Filtering and sorting data.
2.  **🔵 Intermediate:**
    *   Joins (`INNER`, `LEFT`, `RIGHT`).
    *   Aggregations (`COUNT`, `SUM`, `AVG`) and `GROUP BY`.
    *   Subqueries and CTEs (Common Table Expressions).
3.  **🔴 Advanced & Integration:**
    *   Window Functions.
    *   Stored Procedures and Functions.
    *   Connecting PostgreSQL via Python (`psycopg2`) in Jupyter.

## 📓 Workflow
- All SQL queries and analyses are executed and documented within **Jupyter Notebooks**.
- Each notebook includes explanatory markdown cells, code execution, and result visualization.

## 🚀 Learning Objectives
- [x] Master core SQL syntax in PostgreSQL.
- [ ] Explore SQLite for lightweight applications.
- [ ] Learn enterprise-level features in SQL Server.
- [ ] Build end-to-end data pipelines using Python and SQL.

## 📂 How to Run
1.  Ensure you have **PostgreSQL** installed and running.
2.  Open the `.ipynb` files in Jupyter.
3.  Configure your database connection details in the setup cells of each notebook.
4.  Run the cells sequentially to explore the queries.
